# Errbit notification step

## Options

* ``host``  (required) YOUR_ERRBIT_HOST
* ``api-key``  (required) YOUR_ERRBIT_API_KEY
* ``ssl``  default false.
* ``environment`` default production.

## Example

```
- 1syo/errbit-notify@0.0.1:
    host: YOUR_ERRBIT_HOST
    api-key: YOUR_ERRBIT_HOST
```
